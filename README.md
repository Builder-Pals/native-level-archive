# native-level-archive

An index of historical Roblox places. This fork maps Roblox place IDs to metadata/RBXL files for Native Legacy and [`Builder-Pals/nl-web-service`](https://github.com/Builder-Pals/nl-web-service).

## [Open Editor](https://builder-pals.github.io/native-level-archive/)

Requires you to have a local clone of the repository!
